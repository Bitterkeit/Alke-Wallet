# Alke-Wallet
Proyecto Alke Wallet - Módulo 2 Front-end


## 📌 Descripción
Alke Wallet es una billetera digital desarrollada como proyecto final del Módulo 2: Fundamentos del desarrollo Front-end.

## 🎯 Características
- Inicio de sesión con validación
- Administración de saldo (depósitos y retiros)
- Envío de dinero a contactos
- Historial de transacciones
- Persistencia de datos en localStorage

## 🚀 Tecnologías utilizadas
- HTML5
- CSS3 (con variables y animaciones)
- Bootstrap 5
- JavaScript (ES6+)
- jQuery 3.7.1

## 📁 Estructura del proyecto
El proyecto está organizado de la siguiente manera:
- "archivos/" - SPA con persistencia en archivos JSON
- "localstorage/" - Páginas separadas con persistencia en localStorage
